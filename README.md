# 아웃소싱 프로젝트 - HMPAH(How Many People Are Here)
## 프로젝트 소개
>아웃소싱 프로젝트를 진행하면서 소셜 네트워크 기반 헌팅 매칭 서비스인 HMPAH를 개발하게 되었습니다!<br>
>**놀고는 싶은데, 다른 이성과 말을 걸기 두렵다구요?**<br>
